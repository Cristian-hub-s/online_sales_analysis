# Online Sales Analysis

Acesta este un proiect pentru gestionarea unui inventar de produse online. Proiectul include clase pentru produse, gestionarea inventarului și coșul de cumpărături.

## Clase:
- `Product`: Reprezintă un produs cu nume, preț și cantitate.
- `ProductManager`: Gestionează produsele din inventar.
- `Cart`: Gestionează coșul de cumpărături al clientului.

## Funcționalități:
- Adăugarea, eliminarea și afișarea produselor.
- Calcularea valorii totale a inventarului și coșului de cumpărături.
